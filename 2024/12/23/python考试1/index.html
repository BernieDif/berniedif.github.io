Python选择题解析

一、单选题

1. 下列选项中,不能作为Python程序中变量名的是？

  • A. N3sum
  • B. for
  • C. m_1
  • D. info2

【正确答案: B】

解析：在Python中，for 是一个关键字，不能用作变量名。

2. 下列关于说法中，选项错误的是________。

  • A. 集合一旦创建就不可以改变其元素
  • B. 元组是一种序列结构
  • C. 可以对列表进行索引或者切片操作
  • D. 列表中的元素可以为不同的数据类型

【正确答案: A】

解析：集合是可变的，可以添加或删除元素。

3. 元组的特征是？

  • A. 有序，可变
  • B. 有序，不可变
  • C. 无序，可变
  • D. 无序，不可变

【正确答案: B】

解析：元组是有序且不可变的数据类型。

4. random库的seed(a)函数的作用是？

  • A. 生成一个[0.0, 1.0)之间的随机小数
  • B. 生成一个k比特长度的随机整数
  • C. 生成一个随机整数
  • D. 设置初始化随机数种子a

【正确答案: D】

解析：seed(a) 函数用于设置随机数生成器的初始种子，以便生成可重复的随机数序列。

5. python3.*版本内存的默认编码方式为？

  • A. ASCII码
  • B. GBK
  • C. UNICODE
  • D. UTF-8

【正确答案: C】

解析：Python 3 使用 Unicode 作为默认编码方式。

6. 下面代码的输出结果是？

  • A. 12.12
  • B. 34.0
  • C. 12
  • D. 34

【正确答案: A】

解析：z.real 返回复数的实部。

7. Python语言提供的3个基本数字类型是？

  • A. 整数类型、浮点数类型
  • B. 整数类型、浮点数类型、复数类型
  • C. 整数类型、二进制类型、浮点数类型
  • D. 整数类型、二进制类型、复数类型

【正确答案: B】

解析：Python 提供三种基本数字类型：整数、浮点数和复数。

8. 在读写文件之前，用于创建文件对象的函数是？

  • A. open
  • B. create
  • C. file
  • D. folder

【正确答案: A】

解析：open 函数用于打开或创建文件并返回一个文件对象。

9. 给出如下代码

可以输出“python”的是？

  • A. print(s[0:7])
  • B. print(s[:–14])
  • C. print(s[–21: –14].lower)
  • D. print(s[0:7].lower())

【正确答案: D】

解析：s[0:7] 提取字符串的前7个字符，然后使用 lower() 方法将其转换为小写。

10. 以下选项中，不是建立字典的方式是？

  • A. d = {1:[1,2], 3:[3,4]}
  • B. d = {‘张三’:1, ‘李四’:2}
  • C. d = {(1,2):1, (3,4):3}
  • D. d = {[1,2]:1, [3,4]:3}

【正确答案: D】

解析：字典的键必须是不可变类型，而列表是可变类型，因此不能作为字典的键。

题目16

题目描述：
f=open(...) 可以打开一个文件，关于 f 的描述错误的是______

选项解析：
A. 执行 m=f 后，m 和 f 同时表示所打开文件

  • 正确。在Python中，变量 m 和 f 都会引用同一个文件对象。

B. f 是一种特殊的 Python 变量，执行 print(f) 时会报错

  • 错误。f 只是一个普通的变量，指向一个文件对象。执行 print(f) 会打印出文件对象的表示（例如 <_io.TextIOWrapper name='filename' mode='r' encoding='UTF-8'>），而不会报错。

C. 是文件对象引用，在程序中表示文件

  • 正确。f 确实是一个文件对象的引用。

D. f.read() 可以一次性读入文件全部信息

  • 正确。f.read() 方法会读取文件的全部内容。

正确答案：B

题目17

题目描述：
关于Numpy的random模块，描述错误的是

选项解析：
A. np.random.random(100) 生成100以内的随机小数。

  • 错误。np.random.random(100) 生成的是100个在 [0.0, 1.0) 区间内的随机小数，而不是100以内的随机小数。

B. np.random.rand(10,5) 生成服从均匀分布的10行5列的二维随机数

  • 正确。np.random.rand(10,5) 生成的是10行5列的二维数组，元素值在 [0.0, 1.0) 区间内均匀分布。

C. np.random.randn(10,5) 生成服从正态分布的10行5列的二维随机数

  • 正确。np.random.randn(10,5) 生成的是10行5列的二维数组，元素值服从标准正态分布（均值为0，方差为1）。

D. np.random.randint(2,10,size=[2,5]) 创建一个最小值不低于2、最大值不高于10的2行5列数组

  • 正确。np.random.randint(2,10,size=[2,5]) 生成的是2行5列的二维数组，元素值在 [2, 10) 区间内随机整数。

正确答案：A

题目19

题目描述：
已知通过 stu=Student('1810101','李晓明') 可以创建一个 Student 类对象并赋给 stu，则创建该对象时所执行的构造方法的形参个数（不考虑不定长参数的情况）为______

选项解析：

  • 构造方法 __init__ 的形参个数可以通过观察实例化对象时传递的参数个数来确定。
  • stu=Student('1810101','李晓明') 传递了两个参数 '1810101' 和 '李晓明'，因此构造方法的形参个数为2（self 不算在内）。

选项：
A. 4
B. 3
C. 1
D. 2

正确答案：D（但题目中给出的选项是D，但要求填写的是正确答案B的序号，这里应该是题目表述的问题，正确答案应为D对应的实际形参个数，即2个，但题目要求填选项序号，这里按题目要求填B表示选项D是正确的选择，即形参个数为2）。

python不支持的数据类型有

选项解析：

填空题

  1. print(1, 2, 3, sep=’:’) 的输出结果为____________。

    • 正确答案: 1:2:3
    • 解析: sep=':' 参数指定了元素之间的分隔符。

  2. 查看变量类型的Python内置函数是________________。

    • 正确答案: type()
    • 解析: type() 函数返回对象的类型。

  3. 以3为实部4为虚部,Python复数的表达形式为___________

    • 正确答案: 3+4j 或 3+4J
    • 解析: Python中复数的表示方法是 real + imag j，其中 j 是虚数单位。

  4. 在Python中__________表达空类型。

    • 正确答案: None
    • 解析: None 在 Python 中代表空值或空对象。

  5. 字典对象的___________方法可以获取指定“键”对应的“值”，并且可以在指定“键”不存在的时候返回指定值，如果不指定
    则返回None

    • 正确答案: get 或 get() 或 get( )
    • 解析: get() 方法用于从字典中获取值，并且可以提供默认值以防键不存在。

  6. del 命令既可以删除列表中的一个元素,也可以删除整个列表。

    • 正确答案: del
    • 解析: del 关键字用于删除对象，可以是列表的单个元素或整个列表。

  7. 表达式 len(range(1, 10)) 的值为___________。

    • 正确答案: 9
    • 解析: range(1, 10) 生成从1到9的整数序列，len() 函数返回这个序列的长度。

  8. Python 3.x语句 print(1, 2, 3, sep=’:’) 的输出结果为____________。

    • 正确答案: 1:2:3
    • 解析: sep=':' 参数指定了元素之间的分隔符。

  9. Python 3.0运算符中用来计算整商的是_________

    • 正确答案: //
    • 解析: // 运算符用于整数除法，返回整商。

  10. 以3为实部4为虚部,Python复数的表达形式为___________

    • 正确答案: 3+4j 或 3+4J
    • 解析: Python中复数的表示方法是 real + imag j，其中 j 是虚数单位。

  11. 字典中每个元素的“键”与“值”之间使用________分隔开

    • 正确答案: :
    • 解析: 在字典中，键和值之间用冒号 : 分隔。

  12. Python 3.x语句 print(1, 2, 3, sep=’:’) 的输出结果为____________。

    • 正确答案: 1:2:3
    • 解析: sep=':' 参数指定了元素之间的分隔符。

  13. del 命令既可以删除列表中的一个元素,也可以删除整个列表。

    • 正确答案: del
    • 解析: del 关键字用于删除对象，可以是列表的单个元素或整个列表。

  14. 字典对象的___________方法可以获取指定“键”对应的“值”，并且可以在指定“键”不存在的时候返回指定值，如果不指
    定则返回None

    • 正确答案: get 或 get() 或 get( )
    • 解析: get() 方法用于从字典中获取值，并且可以提供默认值以防键不存在。

  15. 在Python中__________表达空类型。

    • 正确答案: None
    • 解析: None 在 Python 中代表空值或空对象。

  16. Python中的列表通过________符号来创建。

    • 正确答案: []
    • 解析: 列表是用方括号 [] 来创建的，例如：my_list = []。

  17. 在Python中，________是用于注释代码的方式。

    • 正确答案: #
    • 解析: 单行注释使用井号 #，例如：# 这是一个注释。

  18. Python中的字符串通过________符号来创建。

    • 正确答案: '' 或 ""
    • 解析: 字符串可以用单引号 ' ' 或双引号 " " 来创建，例如：my_string = 'Hello, World!'。

  19. 在Python中，________是用于多行注释的方式。

    • 正确答案: ''' ''' 或 """ """
    • 解析: 多行字符串或文档字符串使用三个单引号 ''' ''' 或三个双引号 """ """ 来创建，例如：'''这是一个多行 注释'''。
