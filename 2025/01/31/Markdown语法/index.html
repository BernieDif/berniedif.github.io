二、列表

  • 无序列表 使用 - 或 * 或 + 作为标记符号。
    • 子项1
      • 子子项1
      • 子子项2
    • 子项2
  • 有序列表 则使用数字加 . 来实现。
    1. 第一项
    2. 第二项
      1. 子项A
      2. 子项B

三、代码块

在 Markdown 中插入代码块有两种方式：行内代码和代码区块。

  • 行内代码 用一对反引号 ` 包围代码片段，比如 print("Hello, World!")。
  • 代码区块 则用三个反引号 包围多行代码，并可以指定语言以获得语法高亮。

四、链接与图片

  • 链接 的格式为 [显示文本](链接地址)，例如：访问荣耀官网。
  • 图片 的格式与链接类似，只是在前面加一个 !，如：。

五、强调与删除线

  • 强调 可以使用 * 或 _ 将文字包裹起来，形成斜体；双 ** 或 __ 则为粗体。例如：斜体、粗体。
  • 删除线 通过在两个 ~ 之间放置文本来实现，如：这是被删除的文本。

六、引用

使用 > 引导引用的文字，可以嵌套多层引用。

这是第一层引用。

这是第二层引用。

这是第三层引用。

七、表格

Markdown 也支持简单的表格语法，通过 | 分隔列，-: 定义表头对齐方式（左对齐、右对齐、居中对齐）。

表头1 表头2 表头3
内容1 内容2 内容3
内容4 内容5 内容6

八、分隔符

可以使用三个或更多的 -、* 或 _ 来创建水平分隔符。
